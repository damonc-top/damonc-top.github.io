关于我

  我是一名游戏开发者，也是一名游戏爱好者。从事游戏开发完全是出于我的兴趣爱好能够与工作相结合，也是我很喜欢的工作！刚上大学那会，安卓手机开始普及诺基亚垂死挣扎，周围同学都在折腾安卓手机玩游戏，其中就有我。当时永恒战士手游特别火，查资料攻略时不经意间翻到了看雪逆向论坛里面的一些帖子，就这样开始去搞破解游戏。搞破解这玩意很多都不懂，什么jar反编译、apk签名、dll静态调试搞得一头雾水。

  后来在老师建议下，开始去关注学习游戏开发方面的东西，当时搞破解知道永恒战士是用unity3d引擎开发的，就这样开始去找各种资料学习起unity引擎。然后毕业就去了北京一直待到新冠疫情爆发，在顽石互动参与开发坦克指挥官、二战风云2、末日幸存者；在威魔纪元维护过永恒战士VR、参与开发口袋工匠与苏醒之路。2020疫情来了上边对北京疫情的封控很严格，解封就提桶跑路去到了成都。

  个人的一个技术博客站点，记录生活、工作、学习中的一些小事，以及一些比较有趣的事情，留点什么东西也挺不错的，偶尔会在这里停留。
